##Usage
1. Clone the repo to your Web root.  
`git clone https://github.com/seadog007/Light-Adv.git`
2. Run Minecraft/run.sh  
`./Minecraft/run.sh`
3. Run Minecraft/json.sh  
`./Minecraft/json.sh`
4. Open `http://yourhost/Light-Adv/Web`
